Reglas de 3 simple

Simple directa:

La regla de tres simple directa se utiliza cuando el problema trata de dos magnitudes directamente proporcionales. Podemos decir que dos magnitudes son directamente proporcionales cuando al multiplicar o dividir una de ellas por un número, la otra queda multiplicada o dividida respectivamente por el mismo número.

¿Qué porcentaje tiene Juana?

Juana tiene que comprarse unas zapatillas que cuestan $20699. En este momento, cuenta solamente con $8722. ¿Qué porcentaje del total ha alcanzado?

Para saber esto, hay que hacer la regla de 3 simple con los datos que tenemos:

A (20699)------ B(100%)

C (8722)------- X

La fórmula sería la siguiente:

x = C . B / A

x = 8722 . 100 / 20699

x = 872200 / 20699

x = 42,14%

¿Cuánto tardan en terminar la casa?

Un grupo de 3 albañiles tarda alrededor de 12 horas en arreglar una habitación.

Si se contratan 3 albañiles más, ¿cuánto tardarían?

Para saber esto, contamos con los datos:

A (3 obreros)------ B(12 horas)

C (6 obreros)------ X

La fórmula sería la siguiente:

x = A . B / C
